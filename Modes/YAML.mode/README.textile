h1. YAML.mode

YAML.mode is a YAML syntax definition for SubEthaEdit & Coda. This is a work in progress so please be patient ;)

h2. Usage

h3. Installation

Simply check out this repo into either <code>~/Library/Application Support/Coda/Modes</code> or <code>~/Library/Application Support/SubEthaEdit/Modes</code>, be sure to name the directory <code>YAML.mode</code>.
